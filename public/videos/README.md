# Hero video assets

Place hero background videos for the homepage in this folder.

Expected filenames used by the app (playlist order):
- `marota-hero.mp4`
- `marota-hero1.mp4`
- `marota-hero2.mp4`

Recommended export settings:
- Resolution: 1920x1080 (or 1280x720 for smaller files)
- Duration: 12-25 seconds, seamless loop
- Frame rate: 24-30 fps
- Video codec: H.264 for MP4, VP9 for WebM
- No audio track (or muted)
- Target size: ideally under 8 MB per file
